Port Forwarding: Wow, much helping the network...

PiNode-DOGE uses default Dogecoin settings. In this way you will stay sync'd with the blockchain and relay transactions whilst enforcing the rules of the network.

If you would like to help the network further you can allow other node users to sync their nodes blockchain from you. The more people that do this the stronger the network becomes.

!If you choose to do this by forwarding port 22556 you must first ensure you have set a new Master password in the Web Terminal settings menu!

Port forwarding is configured in your router settings and so will be different for each user.
